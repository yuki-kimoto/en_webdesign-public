要素セレクタ - CSS入門【CSS3対応】

要素セレクタは、要素をタグ名を指定するためのものです。

要素セレクタはタグ名と同じです。 

span {
  color:red;
}

要素セレクタを使ってみましょう。

トマト 

<style>
  span {
    color:red;
  }
</style>

<span>トマト</span>

クラスセレクタと要素セレクタの組み合わせ

クラスセレクタと要素セレクタを組み合わせることもできます。

  • トマト
  • レタス
<style>
  li.active {
    color:red;
  }
</style>

<ul>
  <li class="active">トマト</li>
  <li>レタス</li>
</ul>
