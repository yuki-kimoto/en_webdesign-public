visited疑似クラス - CSS入門【CSS3対応】

visited疑似クラスは、訪問済みリンクのスタイルを設定するための疑似的なクラスです。aタグと組み合わせて使われます。

訪問済みリンク 

<style>
  a:visited {
    color:red;
  }
</style>

<a href="/">訪問済みリンク</a>

元のリンクの色と同じ色にするには?

CSSを複数まとめて指定するか、inheritを使います。 

  /* CSSを複数まとめて指定 */
  a, a:visited {
    color:red;
  }
  
  /* inheritを使って、色を継承 */
  a {
    color:red;
  }
  a:visited {
    color:inherit;
  }
