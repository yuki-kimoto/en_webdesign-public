[HTML5] ol tag-Create a numbered list

You can use the ol tag to create a numbered list. 

<ol> </ol>

To add an item to the numbered list, place the li tag inside the ol tag .

  1. Item 1
  2. Item 2
  3. Item 3 
<ol>
  <li> Item 1 </li>
  <li> Item 2 </li>
  <li> Item 3 </li>
</ol>
